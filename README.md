# 🛡️ CryptoShield

## Secure • Encrypt • Protect

CryptoShield is a modern **multi-encryption web application** designed to provide secure text encryption and decryption using both classical and modern cryptographic algorithms.

The project uses the **Web Crypto API** for real browser-based cryptography and provides an interactive cyber-security themed interface.

---

# 🚀 Project Overview

In today's digital world, protecting sensitive information is important.

CryptoShield provides a simple and powerful platform where users can encrypt and decrypt text using different encryption techniques from a single dashboard.

The application combines:

- Modern encryption standards
- Classical cryptography
- Secure browser APIs
- Responsive cyber-security UI

# ⚡ Key Features


## 🔑 RSA Key Generation

Generate secure RSA public and private key pairs using the browser's built-in cryptography engine.


## 🔒 AES Secure Encryption

Uses AES-GCM encryption with:

- PBKDF2 key derivation
- SHA-256 hashing
- Random initialization vector


## 📋 Copy Encryption Output

Quickly copy encrypted or decrypted results with one click.


## 📥 Download Result

Save encryption results as a text file.


## 🧹 Clear Workspace

Reset all fields instantly.


## 📊 Status Monitoring

Real-time operation feedback:

```

✅ Encryption Successful

🔑 RSA Key Generated

❌ Encryption Failed
```

# 📖 How To Use


## Encryption


1. Select an encryption algorithm

2. Enter your message

3. Enter secret key if required

4. Click Encrypt

5. Copy or download the result


## Decryption


1. Paste encrypted text

2. Enter required key

3. Click Decrypt

# 🔐 Security Details


## AES-GCM

AES provides secure symmetric encryption where the same key is used for encryption and decryption.


## RSA-OAEP

RSA provides asymmetric encryption using:

- Public Key
- Private Key
- 2048-bit encryption


## PBKDF2

Password-based key generation using:

- 100000 iterations
- SHA-256 hashing
