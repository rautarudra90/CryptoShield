A modern web-based encryption application that allows users to encrypt and decrypt text using multiple cryptographic techniques. The project features a clean, responsive user interface with support for several popular encryption and encoding methods.

---

## 📖 Overview

CryptoShield is designed to demonstrate the fundamentals of cryptography through an interactive web application. It includes symmetric encryption, classical ciphers, encoding techniques, and a simple RSA demonstration.

This project is suitable for:

- Academic demonstrations
- Cryptography learning
- Mini project submissions
- Portfolio projects
- GitHub showcase

---

## ✨ Features

- 🔒 AES Encryption & Decryption
- 🔑 RSA Demonstration
- 🔤 Caesar Cipher
- 🔄 ROT13 Cipher
- 📦 Base64 Encode & Decode
- 💻 Binary Text Converter
- 🎨 Modern Glassmorphism UI

## 💡 How to Use

1. Open the application.
2. Click **Get Started**.
3. Choose an encryption algorithm.
4. Enter your text.
5. Enter the secret key (if required).
6. Click **Encrypt** or **Decrypt**.
7. View the result in the output box.
