A modern web-based encryption and decryption application that demonstrates multiple cryptographic techniques through an interactive and responsive user interface.

CryptoShield provides a simple platform to understand encryption concepts including **AES, RSA, Caesar Cipher, ROT13, Base64 encoding, and Binary conversion**.

---

# 📌 Project Overview

CryptoShield is an educational cryptography project developed using **HTML, CSS, and JavaScript**.

The application allows users to encrypt and decrypt messages using different encryption algorithms while providing a clean cybersecurity-inspired interface.

This project focuses on:

- Cryptography fundamentals
- Data protection concepts
- Client-side encryption techniques
- Web development practices
